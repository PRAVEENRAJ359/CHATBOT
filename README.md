# Intelligent Enterprise Assistant

## Overview

**Intelligent Enterprise Assistant** is an AI-powered enterprise chatbot platform developed for **Smart India Hackathon (SIH1706)**. The platform is designed to enhance organizational efficiency by providing intelligent assistance for **HR policies**, **IT support**, **company events**, and other internal organizational services using **Natural Language Processing (NLP)** and **Deep Learning** techniques.

The system also integrates document intelligence capabilities including **text extraction**, **document summarization**, and **keyword identification** to improve information accessibility and organizational productivity.

## Workflow

### Step 1: User Authentication

Users securely access the platform using login credentials and Email-based Two-Factor Authentication.

### Step 2: Query Submission

Employees submit queries related to:

- HR Policies
- IT Support
- Company Events
- Organizational Information

### Step 3: AI Processing

The NLP engine processes the query, identifies intent, and performs response generation.

### Step 4: Response Generation

The chatbot provides context-aware responses to employee requests.

### Step 5: Document Processing

Uploaded documents undergo:

- Text Extraction
- Summarization
- Keyword Identification

### Step 6: Monitoring and Analytics

Administrative dashboard tracks system activities and performance metrics.
